# Getting Started
## The Shell
This book is based on the Bash shell. MacOS is using the Zsh shell.
Prompt is $ in Bash and % in Zsh. When logged in as root user the prompt is a #.

To find out what shell you are using run the command:
```bash
echo $SHELL
```
## How to the use the Command Line
Linux and MacOS has a terminal that comes with the OS.
